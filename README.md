# TestNG Practice Repository

This repository contains practice programs and examples created to understand software testing concepts using TestNG.  
It focuses on learning automated testing basics, test execution, and project structure using Java and Maven.

## Topics Covered
- TestNG framework basics  
- Automated test case creation  
- Test execution using XML configuration  
- Maven project structure  
- Assertions and test annotations  
- Basic testing workflows  

## Technologies Used
- Java  
- TestNG  
- Maven  

## Purpose
This project is intended for learning and practicing software testing concepts and improving automation testing skills.
